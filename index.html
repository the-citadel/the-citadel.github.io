The Citadel Framework

  • Headings
  • Fonts
  • Colors

The Citadel Framework establishes base styles for all websites owned by The Citadel. This Framework complies with the college's Brand Guidelines and is meant to demonstrate the proper usage of these guidelines on Citadel websites. Websites may contain items that are not mentioned in the standard brand guidelines and should then default to the framework outlined below. Anything outlined below that does not directly follow the standard brand guidelines has been approved by the brand manager.

With most of The Citadel websites the styles are generated for you which means you can focus on adding the content and the website will do the rest. This page shows how those styles are applied to the content for your reference.

Headings

Headings provide structure for a website. It creates a hierarchy that makes the content easier to read and understand. Below are the headings for our website and use cases for each.

Heading 1 (H1)

A heading 1 should only be used once on each page. This tells search engines as well as the user what the page is about. For almost all Citadel websites the heading 1 is automatically created for the page and, therefore, another one should not be added into the content of the page.

Heading 2 (H2)

Heading 2 should be the highest level of hierarchy for your page structure because Heading 1 is typically generated for us and should only be used once. Heading 2 can be used in any quantity as long as it follows a logical structure.

Heading 3 (H3)

Should follow an H2, another H3, or anything below it (H4, H5, or H6).

Heading 4 (H4)

Should follow an H3, another H4, or anything below it (H5 or H6).

Heading 5 (H5)

Should follow an H4, another H5, or anything below it (H6).

Heading 6 (H6)

Should follow an H5 or another H6.

It might make it easier to picture headings in a list structure to understand the hierarchy:

  • Heading 1: automatically generated for the page, only one per page.
    • Heading 2: the highest level for page content.

Fonts

Poynter Old Style Display

The Citadel uses Poynter Oldstyle Display for all headings on the website (like the one above). For all other elements of text (like this paragraph) The Citadel uses the font Open Sans. There are very few instances where exceptions are made.
